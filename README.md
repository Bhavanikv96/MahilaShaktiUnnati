# Mahila-Shakti Unnati

Android Kotlin app for Women Self-Help Group micro-finance management.

## Included

- Splash screen and demo PIN login
- Dashboard with member, savings, and loan totals
- Member registration and member list
- Savings tracking with paid/pending status
- Loan issue, simple interest calculation, active-loan duplicate prevention, and repayment marking
- Report generation and Android share intent support
- Offline Room database storage

## Tech Stack

- Kotlin
- XML layouts with ViewBinding
- Room Database
- RecyclerView
- Material Components

## Run

1. Open this folder in Android Studio.
2. Let Gradle sync.
3. Run the app on an emulator or Android device.
4. Use demo PIN `1234` on the login screen.

The debug build was verified with:

```powershell
gradle --no-daemon assembleDebug
```
